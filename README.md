# BeReal Clone

A pixel-perfect React Native clone of the BeReal app, built from the provided screenshots.

## Screens Implemented
- Lock screen notification with countdown
- Home feed with My Friends / Friends of Friends / Discovery tabs
- My BeReal. (own post) with tagged friends modal
- Comments screen
- Post creation screen with privacy / location / tagging controls
- Post completion screen
- Tag friends grid with multi-select

## Run Locally

```bash
npm install
npm start
```

Then scan the QR code with the Expo Go app, or press `w` for web / `i` for iOS / `a` for Android.

## Tech Stack
- Expo SDK 57
- React Native 0.86
- TypeScript
- React Navigation
- `@expo/vector-icons`
